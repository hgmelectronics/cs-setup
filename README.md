# CS Setup Desktop App Releases

This repository contains release artifacts and deployment configurations for the CS Setup desktop application.

## Overview

This repository manages the distribution of pre-built binaries and installation packages for multiple platforms including auto-updates.

## Supported Platforms

- **macOS**: `arm64.dmg` installer and `.zip` archive
- **macOS**: `x64.dmg` installer and `.zip` archive
- **Windows**: `.exe` installer and `.zip` archive

## Latest Release

Download the latest version from the [Releases](../../releases) section.

### Auto-Updates

1. Open your existing app version on desktop.
2. If there is a new update you will be prompted to download now or later.
3. Once download is complete you will be prompted to restart the app to access the updated version.

### Installation

1. Go to the [Releases](../../releases) page
2. Download the appropriate installer for your platform:
   - **macOS**: `arm64.dmg` installer and `.zip` archive
   - **macOS**: `x64.dmg` installer and `.zip` archive
   - **Windows**: `.exe` installer and `.zip` archive
3. Run the installer and follow the setup instructions
